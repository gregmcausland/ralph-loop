# Example: Spec File Format

> Delete this file before running the planning phase.

## Job to be done
One sentence: what problem does this solve for the user?

## Scope
- Specific behaviour A
- Specific behaviour B
- Specific behaviour C

## Out of scope
- Thing that belongs in a different spec
- Thing to do later

## Constraints
- Must use existing X (don't rewrite it)
- Must not break Y
- Environment variable Z is available
